「Linux」命令 $cat

发表于 2016-12-27  |  2016-12-28   |   分类于 Linux   |     |   阅读次数

cat是 concatenate (连续)的缩写，用来显示文件内容和标准流上的处理

¶参数

-A, --show-all
-b, --number-nonblank 对非空输出行编号
-E, --show-ends 在每行结束处显示$
-n, --number 由1开始对所有输出的行数编号
-s, --squeeze-blank 有连续两行以上的空白行，就代换为一行的空白行

¶应用

# 显示整个文件内容
$cat filename
# 创建file1(只能创建新文件，不能编辑已有文件)
$cat > file1    
# 将file1和file2的内容加上行号后 附加到新文件file3中
$cat -b file1 file2 > file3
