「Linux」命令 $ls

发表于 2016-12-22  |  2016-12-28   |   分类于 Linux   |     |   阅读次数

ls是 list segment 的缩写(原装不是命令),用于列出指定路径下目录及文件

¶常用选项

  • -l (long) 长格式，显示文件类型、权限、硬链接的数目、文件拥有者、文件所在的组、大小、日期和文件名
  • -F 在不同类型的文件的文件名结尾追加一个字符以示区别。可执行文件后加**“”，目录后加*“/”，管道文件后加“|”，套接字文件后加“=”**，普通文件没有后缀
  • -a (all) 显示所有文件，包括以.开头的文件名(默认不显示)和隐藏文件
  • -A 同*-a*，但不列出**“.”(表示当前目录)和“…”(表示当前目录的父目录)**
  • -R (recursive) 迭代显示目录下所有的子目录。
  • -d (directory) 显示目录本身的信息，而不是列出目录下的文件
  • -t (time) 以文件修改时间排序
  • -1 每行只列出一个文件,垂直显示列表

¶好玩的应用

  1. 计算当前目录下的文件数和目录数
