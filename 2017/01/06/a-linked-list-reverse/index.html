「算法」链表反转

发表于 2017-01-06  |  2017-01-06   |   分类于 算法   |     |   阅读次数

题目：将给定的链表反转（忘记数组）。

¶思路

反转链表，把第一放最后，第二个放倒数第二个，以此类推。就像洗一摞碗，先将最上面的碗拿出来洗了，放到旁边做为新的一摞，然后再从旧的里拿出一个洗后摞到刚才那个上面，等旧的一摞洗完了，新的一摞碗就是旧的反转后的（操作一下）。
为什么不先拿最后一个呢，试试一摞碗（6个以上）把上面的一堆拿起来抽出最后一个，有大量时间，不怕费劲当然可以这么做，不过在我的洗碗生涯里还没有见过这样做的。

步骤:
第一步，确定链表指针。找不到碗就不用洗了，咱也不能只做个空动作。
第二步，拿到第一个数据，放到新链表最后。刚洗好的碗肯定放到已经洗好的最上面了。
第三步，重复第一、二步。把碗洗完了就结束了。

¶用到的算法和数据结构

  • 递归（其中一种方法）
  • 链表

¶收获

  1. 对于链表结构的理解
  2. 分析步骤，把步骤写出来，比用脑子想更好
  3. 确定需求及给定的条件再解题

¶写程序可能会遇到的bug

  1. 指向链表的指针，可能是多个元素，不要把链表看成单一元素
  2. 递归要遵循递归的原则
  3. 退出条件要明确
