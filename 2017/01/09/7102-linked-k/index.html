「算法」链表中倒数第k个数

发表于 2017-01-09  |  2017-01-09   |   分类于 算法   |     |   阅读次数

题目：输入一个链表，输出该链表中倒数第k结点。

¶思路

倒数第k个结点 ，k不能为0,k为0不符合一般的思维的，虽然编程就是一件不符合常规思维的。

  • 第一种，数组，将数据存放在数组里，倒数第k个，只需要找到（count - k）位置的元素；
  • 第二种，栈，将链表元素全部入栈，倒数第k个就是第k个出栈的元素。
  • 第三种，使用参照点point。将参数点point移动到k的位置，即向k-1个指针，此时point和head相差k-1个距离，然后再同时移动参数点point和head，当point走到链表结尾时，head所在的位置就是倒数第k个结点。
    三种方法都需要O(N)的时间复杂度，第一种和第二种需要额外的空间，而第三种则不需要。

¶收获

  1. 对空值的判断，要计算head需要符合条件，否则也不需要继续；
  2. 尝试设置参照点去解决一些类似问题
  3. 测试，验证代码的正确性

¶写程序可能会遇到的bug

  1. 参照点的选取，point和head相差k-1，算是point，那么head就是第k个数
  2. 链表长度小于k的处理
